Introduction

Extremely Annotational RDF Markup (EARMARK) is a meta-syntax for non-embedded markup that can be used for stand-off annotations of textual content with fully W3C-compliant technologies. This document represents the EARMARK ontology definition of the ghost classes, i.e., all those basic and abstract concepts that introduce the object of discourse of this meta-markup language.
